# AniFocus - Anime & TV Series Tracker

## Project Overview
This is a web-based application to manage anime watchlists and track episode progress. Built as a Capstone Project for the Web Development Internship.

## Features
- **Admin Panel:** Add new shows with image URLs and episode counts.
- **Progress Tracking:** Increment episode counts with a single click.
- **Categories:** Filter by All, Watching, Completed, On Hold, Dropped, and Plan to Watch.
- **Data Persistence:** Uses LocalStorage to save data without a backend server.
- **Dark Mode:** User-friendly dark interface.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run
1. Download the repository.
2. Open `index.html` in any web browser.
